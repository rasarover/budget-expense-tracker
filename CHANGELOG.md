# Changelog

All notable changes to BudgetPro will be documented here.

## [v1.0.0] — 2026-05-01

### Added
- Budget Planner with custom categories and subcategories
- Expense tracker with payment methods and filters
- Income tracker with multiple income sources
- Investment tracker with country-specific options (9 countries: India, USA, UK, Australia, Canada, Germany, Singapore, UAE, Other)
- India-specific schemes: PPF, NSC, Sukanya Samriddhi Yojana, NPS, SCSS, KVP, and more
- Dynamic custom investment/category creation
- Demo data loader
- Info (i) tooltips throughout the app
- User authentication — register, login, logout
- Per-user data isolation (each account's data is separate)
- SHA-256 password hashing via Web Crypto API
- Dark / light mode toggle
- Dashboard with Chart.js charts (donut, bar, line)
- Marketing landing page
- Responsive layout with mobile sidebar
